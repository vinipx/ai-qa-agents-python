# Introduction to QORE

Welcome to **QORE** (Quality Core), an enterprise-grade agentic framework designed to orchestrate the entire Quality Engineering lifecycle.

QORE transforms high-level requirements into actionable test plans, automation scripts, and technical documentation using a multi-agent system powered by LangGraph.

:::info Vision
To eliminate manual overhead in Quality Engineering by delegating complex SDLC tasks to specialized AI agents, ensuring 100% traceability and precision.
:::

## Core Value Proposition

- **Autonomous Workflows:** specialized agents for Manual QA, Automation Architecture, Engineering, Code Quality, and Technical Writing.
- **Human-in-the-Loop (HITL):** Integrated approval gates to ensure AI outputs align with business requirements.
- **Real-Time Observability:** A cyber-themed Command Center to monitor agent "thoughts" and graph transitions.
- **Persistent State:** High-performance SQLite checkpointers to maintain system integrity across sessions.

## The Agentic Workforce

QORE features a team of specialized agents:

1.  **Supervisor:** The SDLC Architect that plans and routes tasks.
2.  **Manual QA Expert:** Generates comprehensive test cases.
3.  **Test Automation Architect:** Designs framework strategies.
4.  **Test Automation Engineer:** Implements automated scripts.
5.  **Code Quality Gate:** Reviews and audits generated code.
6.  **Technical Writer:** Updates documentation and READMEs.
